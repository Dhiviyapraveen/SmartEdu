SmartEdu

Gamified Learning Platform for Emerging Technologies (Grades 6–9)

Overview

SmartEdu is a web-based learning platform that makes education interactive using gamification and AI. It uses a Large Language Model (Groq) to generate lessons and quizzes based on student interests.

Features
AI-generated lessons and quizzes
XP, badges, and level system
Mini-games for learning
Progress tracking dashboard
Leaderboard
Tech Stack

Frontend: React.js
Backend: Node.js, Express.js
Database: MongoDB
AI: Groq (LLM)

How it Works:
User logs in
Selects a topic
AI generates lessons
User learns and plays games
Completes quizzes
Gets XP and progress updates


Backend
cd backend
npm install
npm start

Frontend
cd frontend
npm install
